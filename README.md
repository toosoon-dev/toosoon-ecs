# TOOSOON ECS

Library providing Entity-Component-System (ECS) classes.

**Credits**: [ecs-lib](https://github.com/toosoon-dev/toosoon-ecs)

## Installation

Yarn:

```properties
$ yarn add toosoon-ecs
```

NPM:

```properties
$ npm install toosoon-ecs
```
